[선택자 우선 순위]

🔰 div{ color:white; background-color:cadetblue; }
기본 설정을 반영합니다.

🔰 .detail{ color:black; }
동일한 유형이라도 구체적인 설정을 반영합니다.

🔰 .order{ background-color: black; }
🔰 .order{ background-color: brown; }
동일한 유형이라면 마지막 설정이 적용됩니다.

🔰 .imp{ color: pink !important; }
🔰 .imp{ color: skyblue; }
!imporant를 통해 중요한 디자인을 우선 설정합니다.

[ 기본 선택자 설정 ]

🔰 *{ color: burlywood; }
전체 선택자로 HTML 문서의 모든 element를 선택합니다.
🔰 h5{ color:chocolate; }
유형 선택자는 태그명을 이용하여 스타일을 적용할 태그를 선택합니다.
🔰 .detail{ color:black; }
클래스 선택자는 클래스가 적용된 모든 태그를 선택합니다.

🔰 #id{ background-color: cornflowerblue; }
id 선택자는 하나만 존재해야 합니다.

[선택자 응용 & 결합자]

🔰 .multi, h6{ color: chartreuse;}
[class="multi"] 선택자 목록은 여러 선택자를 동시에 선택하도록 한다.
[h6] h6 유형 선택자, multi 클래스 선택자 동시에 사용 가능!
🔰 .parent-one .child-one,
.parent-one #child-two,
.parent-one #child-child{color:gold;}
[id="child-one"] 자손 결합자를 통해 자식 노드 선택 가능
[class="child-two"] 자식이 여럿일 경우 ','로 중복 사용 가능
[id="child-child"] 직속 상속이 아니더라도 사용가능 parent-> child-> child-child

🔰 #parent-two > .child-direct{ background-color: olive; }
[class = "child-direct"] 자식 결합자는 바로 아래 자식인 노드를 선택합니다.
[ parent-two -> (div) -> child-direct] 자손 노드는 해당되지 않습니다.

🔰 .sibling-gen .big-bro ~ .little-bro{ background-color:lightblue }
일반 형제 결합자는 부모를 공유하는 두 번재 요소를 모두 선택합니다.
[little-bro] 단, 뒤따라 오는 요소에만 적용됩니다.
[class=big-bro] 첫번째 요소
[little-bro] 두번째 요소
[little-bro] 두번째 요소

🔰 .sibling-adj .big-bro + .little-bro{ background-color: darkmagenta; }
인접 형제 결합자는 부모를 공유하는 두 번째 요소를 하나 선택합니다.
[little-bro] 단, 뒤따라 오는 요소에만 적용됩니다.
[class=big-bro] 첫번째 요소
[little-bro] 두번째 요소
[little-bro] 두번째 요소
